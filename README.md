## Dataset
Dataset Name: Iris Dataset

## Problem Description
This dataset contains measurements of iris flowers such as sepal length, sepal width, petal length, and petal width.

This is a classification problem.

The target variable is species.

The goal is to build a model that can classify the type of iris flower based on its features.
